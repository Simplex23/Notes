C#:
-----------------------------------------------------
Variabelen:
int     = getallen op geheel
float   = getallen met 8 decimalen
long    = getallen met 32 decimalen
string  = een stukje tekst

Functions:
Console.WriteLine(string);  = toont de string op beeld
Console.ReadLine();         = pauze na WriteLine voordat de programma verder loopt/afsluit.
